Help

My Tasks

In My Tasks you add tasks that you do not want to forget. The tasks will autopopulate with "I need to" before the inputted task. You can clear all the tasks by clicking "Clear All." Clearing the tasks will journal all of the tasks. You must add a task before you take a quiz.

Quiz

The Quiz allows you to rehearse the tasks you need to do for the day. Upon getting a correct answer, your streak will increase by one point. If you get a wrong answer, the streak will reset to zero. You cannot take a quiz unless you have added a task already.

Journal

The Journal allows you to remember tasks that you have cleared because you are done rehearsing for the day. If you do not want a task to be journaled, you can remove them with the "x" button.

Streak

The Streak will increase by one point if you get a correct answer on the quiz. It will reset to zero if you get a wrong answer.
